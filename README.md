## How to Use

1. **Clone** this repository
2. Open `index.html` in your browser (or serve locally with any HTTP server)
3. Use the **VI/EN** toggle to switch between Vietnamese and English

## Local Development

```bash
# Clone
git clone https://github.com/minhduc21092001/My-work-experience.git

# Navigate
cd My-work-experience

# Open in browser
start index.html
```

No build tools required — pure HTML, CSS, and JavaScript.
