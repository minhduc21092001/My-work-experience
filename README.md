## Live Site

The site is built as a bilingual (English / Vietnamese) static website and hosted via **GitHub Pages**.

## Features

- 🌓 **Dark/Light theme toggle**
- 🌐 **Bilingual** (English / Vietnamese) with VI/EN switch
- 📱 **Responsive design** — works on desktop, tablet, and mobile

## How to Use

1. **Clone** this repository
2. Open `index.html` in your browser (or serve locally with any HTTP server)
3. Use the **VI/EN** toggle to switch languages
4. Use the **🌙/☀️** toggle to switch themes

## Local Development

```bash
# Clone
git clone https://github.com/minhduc21092001/My-portfolio.git

# Navigate
cd My-portfolio

# Open in browser
start index.html
```

No build tools required — pure HTML, CSS, and JavaScript.
