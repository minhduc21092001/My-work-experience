Konica Minolta AccurioPress C5080/C5070

Project Overview

The AccurioPress C5080/C5070 is Konica Minolta's digital printing system, a high-speed, production-grade color digital press designed for commercial printing, in-plant, and print-for-pay environments.

This industrial printer offers enhanced paper handling capabilities, advanced automation features, and seamless integration with Konica Minolta's AccurioPro software suite for streamlined print workflows.

Key Features

  • Long paper support (up to 900mm)
  • Intelligent Quality Optimizer (IQ-601)
  • Automated media sensing
  • Envelope printing optimization
  • AccurioPro software integration
  • High-speed color production printing

Role: Embedded Software Engineer (Contractor via FPT Software)

Period: October 2023 – February 2026

Team size: 5

Team: Cross-cultural team with Japanese and Vietnamese engineers

My Responsibilities

  • Developed embedded firmware in C for industrial printers, implementing control logic for electromechanical components including motors, rollers, clutches, solenoids, and high-voltage outputs.
  • Built and maintained core software modules like paper transport, toner supply, image formation, and system calibration — ensuring consistent print quality and stability across product lines.
  • Implemented comprehensive unit tests (using gTest) and integration tests to validate machine behavior against functional specifications, ensuring early defect detection and high code reliability.
  • Debugged complex firmware issues through system log analysis and on-device testing, improving system reliability across release cycles.
