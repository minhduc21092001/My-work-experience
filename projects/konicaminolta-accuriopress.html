Konica Minolta AccurioPress C5080/C5070

Project Overview

The AccurioPress C5080/C5070 is Konica Minolta's latest digital printing system (successor to the AccurioPress C4080 series), launched in January 2026. It is a high-speed, production-grade color digital press designed for commercial printing, in-plant, and print-for-pay environments.

This industrial printer offers enhanced paper handling capabilities (up to 900mm long sheets), advanced automation with the Intelligent Quality Optimizer (IQ-601), automated media sensor for paper type/thickness detection, and seamless integration with Konica Minolta's AccurioPro software suite for streamlined print workflows.

Role: Embedded Firmware Developer (Contractor via FPT Software)

Team: Worked in a cross-cultural team with Japanese and Vietnamese engineers

My Responsibilities

  • Developed embedded firmware in C for 3 industrial printer models, implementing control logic for electromechanical components including motors, rollers, clutches, solenoids, and high-voltage outputs.
  • Built and maintained core software modules for paper transport, toner supply, image formation, and system calibration — ensuring consistent print quality and stability across product lines.
  • Wrote unit and integration tests using GoogleTest to verify module behavior against functional specifications, catching defects early in the development cycle.
  • Debugged complex firmware issues through system log analysis and on-device testing, improving system reliability across release cycles.
  • Collaborated with Japanese hardware and system engineers to integrate firmware with mechanical and electrical subsystems, ensuring smooth interoperability.
  • Participated in code reviews and maintained documentation for firmware modules, contributing to knowledge sharing within the team.

Key Achievements

  • Successfully delivered firmware for 3 printer models that passed rigorous quality assurance and shipped to customers.
  • Implemented automated calibration routines that reduced manual tuning time by improving first-pass yield.
  • Contributed to system stability improvements through systematic root-cause analysis and targeted fixes.
  • Developed reusable test frameworks using GoogleTest that improved regression testing efficiency.
