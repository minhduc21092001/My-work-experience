Konica Minolta AccurioPress C5080/C5070

Project Overview

The AccurioPress C5080/C5070 is Konica Minolta's digital printing system, a high-speed, production-grade color digital press designed for commercial printing, in-plant, and print-for-pay environments.

This industrial printer offers enhanced paper handling capabilities, advanced automation features, and seamless integration with Konica Minolta's AccurioPro software suite for streamlined print workflows.

Role: Embedded Software Engineer (Contractor via FPT Software)

Period: October 2023 – February 2026

Team size: 5

Team: Cross-cultural team with Japanese and Vietnamese engineers

My Responsibilities

  • Developed embedded firmware in C for industrial printer models, implementing control logic for electromechanical components including motors, rollers, clutches, solenoids, and high-voltage outputs.
  • Built and maintained core software modules for paper transport, toner supply, image formation, and system calibration — ensuring consistent print quality and stability across product lines.
  • Wrote unit and integration tests using GoogleTest to verify module behavior against functional specifications, catching defects early in the development cycle.
  • Debugged complex firmware issues through system log analysis and on-device testing, improving system reliability across release cycles.
  • Collaborated with Japanese hardware and system engineers to integrate firmware with mechanical and electrical subsystems, ensuring smooth interoperability.
  • Participated in code reviews and maintained documentation for firmware modules, contributing to knowledge sharing within the team.

Key Achievements

  • Successfully delivered firmware for printer models that passed rigorous quality assurance and shipped to customers.
  • Contributed to system stability improvements through root-cause analysis and targeted fixes.
  • Developed reusable test modules using GoogleTest that improved regression testing efficiency.

Technologies Used

C FreeRTOS GoogleTest Git / GitLab Jira

Tổng Quan Dự Án

AccurioPress C5080/C5070 là hệ thống máy in kỹ thuật số của Konica Minolta, máy in màu tốc độ cao, chất lượng sản xuất, được thiết kế cho môi trường in thương mại.

Máy in công nghiệp này cung cấp khả năng xử lý giấy nâng cao, các tính năng tự động hóa, và tích hợp liền mạch với bộ phần mềm AccurioPro để tối ưu quy trình in ấn.

Vai trò: Kỹ Sư Phần Mềm Nhúng (Contractor qua FPT Software)

Thời gian: Tháng 10/2023 – Tháng 02/2026

Quy mô đội: 5

Đội ngũ: Đa văn hóa với kỹ sư Nhật Bản và Việt Nam

Trách Nhiệm Của Tôi

  • Phát triển firmware nhúng bằng C cho các dòng máy in công nghiệp, điều khiển các thành phần điện-cơ bao gồm motor, roller, clutch, solenoid và đầu ra cao áp.
  • Xây dựng và bảo trì các module phần mềm cốt lõi cho vận chuyển giấy, cấp mực, tạo hình ảnh và hiệu chuẩn hệ thống — đảm bảo chất lượng in ổn định qua các dòng sản phẩm.
  • Viết unit test và integration test sử dụng GoogleTest để kiểm tra hoạt động của module theo đặc tả chức năng, phát hiện lỗi sớm trong chu kỳ phát triển.
  • Debug các vấn đề firmware phức tạp thông qua phân tích system log và kiểm tra trên thiết bị thực, cải thiện độ tin cậy hệ thống qua các bản phát hành.
  • Hợp tác với kỹ sư phần cứng và hệ thống người Nhật để tích hợp firmware với các subsystem cơ khí và điện tử.
  • Tham gia code reviews và duy trì tài liệu cho các module firmware.

Thành Tựu Chính

  • Bàn giao firmware thành công cho các dòng máy in vượt qua kiểm định chất lượng và xuất xưởng.
  • Đóng góp vào cải thiện độ ổn định hệ thống thông qua phân tích nguyên nhân gốc rễ và sửa lỗi có mục tiêu.
  • Phát triển các module kiểm thử tái sử dụng dùng GoogleTest, nâng cao hiệu quả kiểm thử hồi quy.
