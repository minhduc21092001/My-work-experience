LG Wall Oven Series

Project Overview

LG Wall Ovens are premium built-in kitchen appliances that combine sophisticated design with advanced cooking technology. The product line includes single, double, and combi (microwave + oven) models, offering features such as ProBake Convection®, EasyClean®, and smart connectivity.

As part of the firmware team at LGEDV (LG Electronics Development Vietnam), I contributed to the development of user interface and board communication firmware that powers the control, display, and connectivity features of these wall oven models.

Role: Embedded Firmware Developer

Team: Worked in a global R&D environment with Korean, Vietnamese, and other international engineers

My Responsibilities

  • Developed user interface (UI) firmware for the wall oven display panels, implementing menu navigation, cooking mode selection, timer/settings management, and real-time status feedback.
  • Implemented board-to-board communication protocols (I²C, UART, SPI) between the main control board, display board, touch panel, and sensor modules.
  • Developed firmware for multiple product variants — single oven, double oven, and combi (microwave + oven) models — managing hardware abstraction and feature configuration across different platforms.
  • Collaborated with hardware engineers to validate board designs, resolve integration issues, and optimize firmware for production.
  • Performed system-level debugging using oscilloscopes, logic analyzers, and serial debug tools to identify and resolve timing, communication, and performance issues.
  • Contributed to software architecture for modular, reusable UI components that accelerated development of new models.

Key Achievements

  • Delivered UI firmware for 3 wall oven platforms (single, double, combi) that met LG's quality standards for global markets.
  • Developed board communication drivers that improved data transfer reliability between control and display modules.
  • Implemented modular software architecture enabling reuse of UI components across multiple product lines, reducing development time for new variants.
  • Successfully resolved complex timing and EMI-related issues during system integration testing, ensuring product reliability.
