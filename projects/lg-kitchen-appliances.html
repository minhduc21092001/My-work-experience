LG Kitchen Appliances UI

Project Overview

As a Research Engineer at LGEDV (LG Electronics Development Vietnam), I develop embedded user interfaces for LG's kitchen appliance lineup, including wall ovens and coffee machines. The work focuses on creating intuitive, responsive, and visually polished UI that meets LG's global product standards.

The UI runs on Linux-based embedded platforms and communicates with the main control board via UART and with the Wi-Fi module via SPI, handling bidirectional data exchange and command processing for real-time appliance control and monitoring.

Role: Research Engineer

Period: March 2026 – Present

Team size: 4

Team: Global R&D environment with Korean, Vietnamese, and international engineers

My Responsibilities

  • Developed embedded UI for kitchen appliances using Qt and LVGL, built and tested on Linux-based environments.
  • Implemented communication interfaces between the display module and other components — UART with the main board and SPI with the Wi-Fi module — handling data exchange.
  • Ensured UI responsiveness, smooth animations, and intuitive user interaction aligned with LG's product design standards.
  • Delivered clear technical documentation and test cases, enabling efficient QE workflows and improving collaboration across cross-functional teams.
  • Leveraged AI tools to develop automation solutions, significantly reducing task completion time and enhancing overall team productivity.
