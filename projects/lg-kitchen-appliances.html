LG Kitchen Appliances UI

Project Overview

As a Research Engineer at LGEDV (LG Electronics Development Vietnam), I develop embedded user interfaces for LG's kitchen appliance lineup, including wall ovens and coffee machines. The work focuses on creating intuitive, responsive, and visually polished UI that meets LG's global product standards.

The UI runs on Linux-based embedded platforms and communicates with the main control board via UART and with the Wi-Fi module via SPI, handling bidirectional data exchange and command processing for real-time appliance control and monitoring.

Role: Research Engineer

Period: March 2026 – Present

Team size: 4

Team: Global R&D environment with Korean, Vietnamese, and international engineers

My Responsibilities

  • Developed embedded UI using Qt for wall oven display panels — implementing menu navigation, cooking mode selection, timer/settings management, and real-time status feedback.
  • Developed embedded UI using LVGL for coffee machine touchscreen interface, delivering smooth animations and intuitive user interaction on resource-constrained hardware.
  • Implemented communication interfaces between the display module and external components — UART with the main control board and SPI with the Wi-Fi module.
  • Ensured UI responsiveness and smooth animations aligned with LG's product design standards and user experience guidelines.
  • Built and tested software on Linux-based development environments, using cross-compilation toolchains and target hardware validation.
  • Produced technical documentation including change lists, design notes, and test cases to support QE and cross-team collaboration.

Key Achievements

  • Delivered Qt-based UI for wall oven platforms that met LG's quality and performance standards.
  • Developed LVGL-based touch UI for coffee machine, optimizing graphics and responsiveness for embedded display hardware.
  • Established UART/SPI communication drivers enabling reliable data exchange between display, main board, and Wi-Fi module.
  • Created comprehensive test cases and technical documentation that improved cross-team handoff and QE validation efficiency.

Technologies Used

C/C++ Qt LVGL Linux UART / SPI Git / GitLab Jira

Tổng Quan Dự Án

Là Kỹ Sư Nghiên Cứu tại LGEDV (Trung tâm Nghiên cứu & Phát triển LG Electronics Việt Nam), tôi phát triển giao diện người dùng nhúng cho các thiết bị nhà bếp LG, bao gồm lò nướng và máy pha cà phê. Công việc tập trung vào tạo ra UI trực quan, phản hồi nhanh và đáp ứng tiêu chuẩn sản phẩm toàn cầu của LG.

UI chạy trên nền tảng Linux nhúng, giao tiếp với bo mạch chính qua UART và với module Wi-Fi qua SPI, xử lý trao đổi dữ liệu hai chiều và điều khiển thiết bị theo thời gian thực.

Vai trò: Kỹ Sư Nghiên Cứu

Thời gian: Tháng 03/2026 – Nay

Quy mô đội: 4

Đội ngũ: Môi trường R&D toàn cầu với kỹ sư Hàn Quốc, Việt Nam và quốc tế

Trách Nhiệm Của Tôi

  • Phát triển UI nhúng dùng Qt cho màn hình lò nướng — menu điều hướng, chọn chế độ nấu, quản lý hẹn giờ/cài đặt, phản hồi trạng thái thời gian thực.
  • Phát triển UI nhúng dùng LVGL cho màn hình cảm ứng máy pha cà phê, tối ưu hoạt ảnh mượt mà trên phần cứng có tài nguyên hạn chế.
  • Triển khai giao tiếp giữa module hiển thị và các thành phần ngoại vi — UART với bo mạch chính, SPI với module Wi-Fi.
  • Đảm bảo UI phản hồi nhanh và hoạt ảnh mượt theo tiêu chuẩn thiết kế sản phẩm LG.
  • Xây dựng và kiểm thử phần mềm trên môi trường Linux, sử dụng cross-compilation toolchain và kiểm tra trên thiết bị thật.
  • Tạo tài liệu kỹ thuật bao gồm change list, design notes và test cases hỗ trợ QE và cộng tác nhóm.

Thành Tựu Chính

  • Bàn giao UI dùng Qt cho lò nướng đáp ứng tiêu chuẩn chất lượng và hiệu suất LG.
  • Phát triển UI cảm ứng dùng LVGL cho máy pha cà phê, tối ưu đồ họa cho phần cứng nhúng.
  • Thiết lập driver giao tiếp UART/SPI cho phép trao đổi dữ liệu tin cậy giữa màn hình, bo mạch chính và module Wi-Fi.
  • Tạo test cases và tài liệu kỹ thuật toàn diện, cải thiện hiệu quả bàn giao giữa các nhóm và kiểm duyệt QE.
